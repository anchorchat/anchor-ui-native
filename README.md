# AnchorUI Native

## Requirements

- [Node](https://github.com/creationix/nvm) ^4.8.0 || ^5.7.0 || ^6.2.2 || >=8.0.0
- [Yarn](https://yarnpkg.com/en/)

## Getting started

Install the node_modules:
```bash
yarn install
```

Copy the `lib` folder to `docs` and `examples`:
```bash
make build
```

Copy the `lib` folder and watch for changes:
```bash
yarn run watch
```

# AnchorUI Native Docs

## Requirements

- [Node](https://github.com/creationix/nvm) ^4.8.0 || ^5.7.0 || ^6.2.2 || >=8.0.0
- [Yarn](https://yarnpkg.com/en/)

## Getting started

Install the node_modules:
```bash
yarn install
```

## Running

Start the development server:
```bash
yarn start
```

## Builds

Create a production build:
```bash
yarn build
```
